## Installation

```bash
$ nvm use 18.19.0
$ yarn install
```

## Set up environment variables

Next env variables are required:

```bash
SHOPIFY_STORE=
SHOPIFY_ACCESS_TOKEN=
```

## Running the app

```bash
# development
$ yarn run start
```

## Swagger

Open http://localhost:3000/api

## Test

```bash
# unit tests
$ yarn run test

```
